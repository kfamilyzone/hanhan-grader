✅ What earns a 4–5

  • Directly simulates FAANG interviews (DSA, system design, behavioral) or helps the Vietnamese teen with English in a US school context
  • Multi-step AI interaction — not a one-shot prompt wrapper
  • Polish: live preview, history, export, voice, comparison features
  • ~3+ hrs of vibe coding visibly present in the product

❌ Avoid

  • One-shot wrappers (Idea Generator, Commit Writer, Comment Generator)
  • Tools that overlap existing agents without distinct added value
  • Generic academic tools not specialized for either persona

💡 High-value builds for #31–#100
